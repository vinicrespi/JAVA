My Order Demo

Aplicação que simula um aplicativo de pedidos de comida.

Projeto desenvolvido para o minicurso de Spring Boot + Java 8 (UDESC-CCT).
Back-End

    Java 8;
    Spring Boot;
    JPA/Hibernate;
    Swagger2;
    ModelMapper;

Front-End

    AngularJS;
    Bowel;
    Yeoman;
    Grunt;
